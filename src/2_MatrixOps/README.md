## Instructions
`./2_Client [options] matrices...`

| Options | Description      |
|---------|------------------|
| mul     | Multipy Matrices |

### Examples
Multiply a 2x2 and a 2x3 matrix  
`./2_Client mul "[[1,2][3,4]]" "[[9,8,7][6,5,4]]"`

Multiply a 4x3 and a 3x2 matrix  
`./2_Client mul "[[1,2,3][9,1,1][3,0,1][1,2,3]]" "[[1,2][3,4][5,6]]"`
